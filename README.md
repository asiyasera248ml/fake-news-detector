# 📰 Fake News Detector

📌 **Description**
This is an AI-based web application that detects whether a news article is **Fake ❌ or Real ✅** using Machine Learning and Natural Language Processing (NLP).

---

## 🚀 Features

* Detect fake and real news instantly
* User-friendly web interface
* Handles real-world noisy datasets
* Uses NLP for text analysis

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* NLP (TF-IDF Vectorization)

---

## ▶️ How to Run

### Install required libraries

pip install streamlit pandas scikit-learn numpy

### Run the app

streamlit run ui.py
